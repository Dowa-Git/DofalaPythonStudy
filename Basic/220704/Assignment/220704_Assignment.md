# 2022/07/04 스터디 연구 과제
## <u>문제</u>
* 양의 정수 N를 입력 받습니다.
* 입력받은 정수 N만큼 반복하여 갯수가 하나씩 늘어나게 *을 출력하세요.
* 첫째 줄에는 * 1개, 둘째 줄에는 * 2개, N번째 줄에는 별 N개가 찍혀야 합니다.
  
## <u>입력</u>
* 첫째 줄에 N을 입력 받습니다. (1 <= N)

## <u>출력</u>
* 첫째 줄부터 N번째 줄까지 차례대로 줄 수에 맞는 *을 출력


## <u>예시 입력 1</u>
```
5
```

## <u>예시 출력 1</u>
```
*
**
***
****
*****
```

## <u>예시 입력 2</u>
```
3
```

## <u>예시 출력 2</u>
```
*
**
***
```
