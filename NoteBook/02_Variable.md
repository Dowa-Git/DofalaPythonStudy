## 변수 (Variable)
***
## 1.변수란?
* 변수는 데이터가 처리 혹은 출력에 앞서 저장되는 공간입니다.
* 입력 시 임의의 값이 대입 될 수 있기 때문에 변수라 칭해집니다.

## 2. 변수의 기본 타입
* #### 숫자형 (Numeric Type)
  * ##### int(integer) : 정수, 소수점이 붙지 않은 숫자들. 
  * ##### float(float) : 실수, 소수점이 있는 숫자들. 정수와 정수 사이를 떠다닌다 하여 float이라 합니다.

    > 정수형, 실수형은 인트, 플롯 이외에도 값의 범위에 따라 매우 다양하나 위의 두 종이 가장 기본입니다.
    >  다른 타입들은 최적화의 이유 혹은 매우 큰 값을 써야 할 때만 사용합니다. 
    
    > 이런 경우들을 설명하기 위해서는 각 숫자 변수 타입들의 범위와  용량, 컴퓨팅 환경에서의 변화 등 기초 내용에서 벗어나 os 단에서의 메모리에 대한 설명이 필요하기 때문에 일단 제외하였습니다.
 
* #### 문자형 (String Type)
  * ##### Char(character) : 내부 값은 사실 정수이나 이 정수 값을 통해 규약에 맞는 문자가 연결 됩니다.

 
    >문자형의 경우 string 타입을 쓰는 게 일반적이나 string은 따로 시간을 할애해야 할 정도로 내용이 방대하기 때문에 지금은 char 타입만 알고 계시면 됩니다.

* #### 논리형 (String Type)
  * ##### Bool(Boolean 논리형) : 참 1, 거짓 0 으로 표기 됨. 파이썬 문법으로는 True, False (대문자 유의)

* #### 이외에도 복소수형, 자료구조형 등 많은 타입들이 있으나, 각각 설명해야 할 게 많아 따로 챕터를 구성하려 합니다.
* 위에 언급한 int / float / char / bool 만 꼭 기억해주세요.

## 3. 변수의 선언
* #### 변수의 선언이란 컴퓨터에게 변수를 사용하겠다고 알리는 것입니다.
* #### 이 과정은 마치 박스에 물건을 포장하고 네임 태그를 붙히는 것과 비슷합니다.
  ```
  # int 타입의 변수 선언
  # NumberZero라는 이름의 변수를 만들고 여기에 숫자 0을 넣어주세요.
  NumberZero = 0

  # float 타입의 변수 선언
  # FloatSix라는 이름의 변수를 만들고 여기에 실수 6.0을 넣어주세요.
  FloatSix = 6.0 

  # char 타입의 변수 선언
  # CharA라는 이름의 변수를 만들고 여기에 문자 A를 넣어주세요.
  CharA = 'A'

  # bool 타입의 변수 선언
  # IsTrue라는 이름의 변수를 만들고 여기에 True 값을 넣어주세요.
  IsTrue = True
  ```
* #### 파이썬은 타 언어와 다르게 변수의 선언 과정이 매우 간단하게 이루어집니다.
* #### 변수의 타입에 대한 명시가 필요 없이 넣은 값에 따라 타입이 변화하기 때문입니다.
