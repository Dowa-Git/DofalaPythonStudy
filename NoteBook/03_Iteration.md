## 반복문 (Iteration)
***
### 1.반복문?
* #### 반복문은 프로세스 안에서 특정 명령을 반복 수행하도록 하는 문법입니다.
* #### 코딩을 통해 기대되는 대부분의 아웃풋이 (자동화된 작업, 예를 들면 주어진 수십, 수백개의 마야씬을 열어 필요한 데이터만을 익스포트 하는 등의) 반복되는 작업 혹은 (실시간 온라인 게임이나 영상 스트리밍 등의)연속되는 작업이기 때문에 반복문은 코드 스크립팅에 있어 매우 중요한 부분을 차지합니다.
* #### 반복문은 특정한 규칙에 따라 반복하며 특정한 데이터를 함께 순회하는 것도 가능합니다.
    > ##### 특정한 데이터란 반복 기능이 내부에 구현 되어 있는 타입의 오브젝트를 말합니다. 이 부분에 대해서는 지금은 그냥 읽어만 두시면 될 것 같습니다.
* #### 반복문은 반복되는 횟수만큼 시간 소요가 중첩됩니다. n의 시간 소요가 있는 작업을 m번 반복한다면 총 시간 소요가 n*m이 되는 것 처럼요. 다중 루프 구조(예를 들면 for 문 안에 for 문 안에 for문이 있는 경우)일 경우 시간 소요는 n제곱 수준으로 늘어나게 됩니다.
* #### 업무 중 혹은 일상에서 컴퓨터 사용 중 겪어 보셨을 '응답 없음' 현상, 프로그램의 일시 정지 현상이 반복문의 시간 소요, 혹은 논리 오류로 인한 무한  반복으로 인해 발생합니다.
    > ##### 모든 무한 루프가 부정적인 것은 아닙니다.  
    > ##### 여러분이 매일 사용하고 계시는 Windows OS, Maya나 Unreal Engine 또는 퇴근 후 즐기는 게임들 모두가 종료를 의미하는 입력(종료 버튼, alt + f4 등)이 들어오기 전 까지 프로그램 코어에서 항상 무한 루프가 진행 되고 있습니다.  
    > ##### 이러한 무한 루프로 인해 우리는 우리가 사용하는 프로그램에 실시간으로 입력(키보드, 마우스, 타블렛 등)을 주고, 출력(모니터 화면의 렌더링 데이터)을 받을 수 있습니다.  
### 2.반복문의 종류
* #### while 반복문
  * ##### while 반복문은 주어진 조건이 '참'일 때에만 반복합니다.
  * ##### 조건이 만족되는 한 무한으로 작업을 반복합니다.
    > ##### anti infinite loop 기능이 있는 환경의 경우에는 제한 횟수 만큼만 반복하지만, 천문학적인 숫자이므로 대부분의 경우를 처리할 수 있으며 제한을 푸는 것이 가능한 환경도 있습니다.  
  * ##### 조건이 명확하지 않은 while 문은 의도하지 않은 무한루프로 연결 되므로 주의하여야 합니다.
   
  * #### Code Example
    ```python
    # Condition 변수 = 참
    Condition = True
    # 반복 [Condition 변수가 참인 동안]
    while Condition:
    #   출력기('*')
        print('*')
    ```
    > ##### 이 경우 Condition 변수는 True(참)이며 변하지 않으므로 무한 루프를 예상할 수 있습니다.
    ```python
    # Counter 변수 = 0
    Counter = 0
    # 반복 [Counter 변수가 10보다 작은 동안]
    while Counter < 10:
        print(f'현재 {Counter} 번 째 반복 중입니다.')
        # Counter 변수 + 1
        Counter += 1
    ```
    > ##### 이 경우 매 반복마다 반복 횟수를 출력하며 Counter 변수가 1씩 증가합니다. 
    > ##### 만약 Counter 변수가  10 이상으로 넘어가게 되면 While 문의 조건에 부합하지 않아 반복이 중단됩니다.  
    > ##### 배우지 않은 문법이 있으므로 지금은 그냥 눈에 익힌다고 생각만 해주세요.  
* #### for 반복문
  * ##### for 반복문은 주어진 범위에 따라 반복합니다.
  * ##### 많은 사용법이 있으나, 현재는 range()만을 다루겠습니다.
  * #### Code Example
    ```python
    # For 반복문
    # 반복 [0부터5까지]:
    for i in range(0,5):
    #   출력기('a')
        print('a')
    ```
    > ##### 이 경우 0부터 4까지 (5가 되면 반복이 중단) 5번 반복하게 됩니다.  
    > ##### 따라서 a가 5번 출력되게 됩니다.
  * ##### for 문은 while 문보다 조건이 명확하여 리스크가 적고 간편하게 쓸 수 있지만 그 만큼 남용(모든 경우의 반복처리를 for문에만 의지)하는 사례 또한 많은 문법입니다.
* #### for문과 while문의 차이와 특징을 꼭 기억하시고 적재적소에 사용하셨으면 좋겠습니다.
